Python百万级数据分析实战2 - 姓名普查

数据源

数据来自hadley/babynames: An R package containing US baby names from the SSA (github.com)
数据量：192万
原作者使用R语言分析，而我选择python。

import列表

本项目需要使用matplotlib进行数据可视化处理。 引用完成后设置绘图后端为plotly。

# import列表
import pyreadr
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

#选择绘图后端
pd.options.plotting.backend = "plotly"
#pd.options.plotting.backend = "matplotlib"

导入数据

使用pyreadr读取rda文件。

#导入数据
result = pyreadr.read_r(r"E:\Coding\python\数据分析\百万级人名统计\babynames-master\data\babynames.rda")
print(result.keys()) ## 查看获取的内容
df = result["babynames"]

起名潮流变迁

# 通过姓名人数降序排序，并提取前1000行
def get_top1000(group):
    return group.sort_values(by='n', ascending=False)[:1000]

grouped = df.groupby(['year','sex'])
top1000 = grouped.apply(get_top1000)
# 后处理并绘图
top1000.reset_index(drop=True, inplace=True)
top1000.year = pd.to_numeric(top1000.year, downcast='integer')
ptable = top1000.pivot_table('prop', index='year', columns='sex', aggfunc='sum')
fig_2 = ptable.plot(title='不同性别前1000的名字变化趋势图')
