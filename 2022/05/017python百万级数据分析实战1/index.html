Python百万级数据分析实战1 - 电影评分

2022-05-13 Coding 数据分析 评论 字数统计:634字 阅读时长:2分

Python百万级数据分析实战1 - 电影评分

数据源

数据来自GroupLens
- 样本数量：1,000,209;
- 数据说明：该数据集为 6000 个用户，对4000部电影的 100 万个评分，分布在用户信息、电影信息和评分信息3个表格。

数据导入

在一切开始之前，别忘了import。

import numpy as np
import pandas as pd

接下来分别导入三个表。

users = pd.read_table("ml-1m/users.dat",sep='::',header=None,
					  names=['userid', 'gender', 'age', 'occup', 'zip'])
movies = pd.read_table("ml-1m/movies.dat",index_col=None, sep='::',
					   encoding = 'cp1252', header=None, 
					   names=['movieid', 'title', 'genres'])
ratings = pd.read_table("ml-1m/ratings.dat", sep='::',header=None,
						names=['userid', 'movieid', 'rating', 'timestamp'])

数据汇总

使用merge函数将三个表汇总到一起得到最终原始数据集。

df = pd.merge(pd.merge(ratings, users), movies)

上述操作使用了两个嵌套的merge函数，也可以拆开写，如下。

pd.merge(ratings, users, on='userid')
pd.merge(ratings, movies, on='movieid')

数据分析

在拿到数据集后，就可以代入一些场景进行分析了。

按性别查看每部电影的平均评分

先考虑一个典型的分类统计需求。这里我们使用两种方法进行分类统计。
两种方法结果相同。

# 使用 groupby + unstack 函数
mean_ratings = df.groupby(by=['title', 'gender']).rating.mean().unstack()
# 使用 pivot_table
mean_ratings = df.pivot_table(index='title', columns='gender', values='rating', aggfunc='mean')

剔除评分人数小于100人的样本

典型的数据清洗需求，剔除样本量过小的数据。

ratings_by_title = df.groupby('title').size()
active_title = ratings_by_title.index[ratings_by_title >= 100]
mean_ratings_active = mean_ratings.loc[active_title]

根据女性评分排序

排序不多说，直接上代码。

female_likes = mean_ratings_active.sort_values('F', ascending=False)

寻找按性别评分分歧最大的电影

这个也好说，直接按性别评分的差值排序即可。

mean_ratings_active['diff'] = abs(mean_ratings_active.F - mean_ratings_active.M)
gender_differences = mean_ratings_active.sort_values('diff')

复杂需求：寻找争议最大的10部电影

争议最大也就是说不同人打分差别很大，反映在统计上就是标准差/方差较大。
因此先算标准差，然后剔除样本量不足的数据，最后排序即可。

ratings_std = df.groupby('title').rating.std()
ratings_std_active = ratings_std.loc[active_title]
top10 = ratings_std_active.sort_values(ascending=False).head(10)

实战总结

  • groupby 分组统计和数据透视表一定程度上是相通的
  • 可以利用 groupby 分组统计结果的索引，来筛选数据
  • 即使是百万计数据，实际运行时间也不超过一秒，pandas 运行效率可见一斑
